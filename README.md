# 这是什么？
这是一个基于留学生调查问卷的资料汇总项目，旨在为准备留学的学生提供参考、减少信息搜寻时间、并降低受骗概率。

# 项目起因
受到[大学生活质量指北](https://colleges.chat/)的启发，同时因为就读外国高校时所需的信息过于鱼龙混杂（如：租房性价比、XX是真的吗 不会是诈骗吧……等），且身在国内的学生大多没有实地考证的能力。   
同时，在申请的过程中有不少需要注意的地方（如：PS文书的书写、面试的准备等），但是实际上这些东西的**有参考价值的版本**基本都要在中介机构**花钱**才能找到。  
申请学校已经让人身心俱疲，各种其他的准备工作更是让人焦头烂额，我们何不利用前人的经验，减少时间成本呢。因此，本站在此提供信息参考平台，为留学生提供帮助。 

# 问卷及交流
这是[问卷链接](https://forms.gle/kjCYUDtaR55mgysS7)。问卷使用Google Forms，国内用户请自备VPN。  

问卷中除为标记问卷所需的必须问题外，均设为可不填写。若您不想提供某项信息或认为涉及隐私，您可以选择不填，但是**请勿填写模糊或道听途说的数据**。您在此提交的数据默认以[署名-非商业性使用-相同方式共享 4.0 国际 (CC BY-NC-SA 4.0)](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh)协议共享给我们。若您所提交学校的数据已存在于我们的信息库中，且无有意义补充，我们将忽略您所提交的数据，非常抱歉。  

询问学校信息、参与项目讨论、聊天请加入[telegram群组](https://www.baidu.com)  （目前还未建立）
