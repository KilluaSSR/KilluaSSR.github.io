环境配置

2023年的今天，Python的配置已经大幅降低难度了，因为环境变量Path什么的都能自动给你配置好，不像我当年什么都得自己敲。

下载并安装Python

  1. 只需要点击开始菜单，在里面找到Windows商店。
  2. 搜索Python，选择最新版（当下最新版是3.12）
  3. 点击安装，静候即可。
  4. 检查是否安装成功：按住Windows徽标键+R，输入cmd。
  5. 在弹出的黑窗口里输入
  1. 若返回了一个安装路径则为安装成功。
  2. 启动Python：输入
  1. 控制台的输入框变为 >> ，运行成功。

安装Pycharm

我们应该使用强大而智能的工具，以提高工作的效率。Pycharm则是各种IDE里（IDE：集成开发环境）中做的最优秀的。它具有强大的代码提示功能。（你会发现这项功能是多么的Essential）

点击此处下载Windows版Pycharm2023.2.5，下载后双击安装即可。

启动后会发现弹出激活步骤。因为这是付费软件。JetBrains的定价是第一年249美元，第二年199美金，第三年149美金。有能力者请支持正版。
当然你也可以选择破解——步骤并不繁琐
